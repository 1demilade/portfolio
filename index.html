Project 1

A smart web-based application that helps farmers detect cocoa plant diseases early using image recognition. Built with Flask and powered by a TensorFlow model (DenseNet121), the system allows users to upload images of cocoa leaves and receive instant predictions of possible diseases. It aims to improve crop health, increase yield, and support sustainable farming by providing fast, accessible, and reliable diagnosis.

Obstacle Avoidance Robot

The Obstacle Avoidance Robot is an autonomous embedded system designed to navigate its environment intelligently by detecting and avoiding obstacles in real time. The system uses ultrasonic sensors to measure distances between the robot and nearby objects, enabling it to make immediate navigation decisions without human intervention.
