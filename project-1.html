Cocoa Disease Classifier

A smart web-based application that helps farmers detect cocoa plant diseases early using image recognition.

Live Link

Project Overview

Cocoa Care is an intelligent, web-based decision support system designed to assist cocoa farmers in the early detection and management of plant diseases. The platform leverages modern machine learning techniques to analyze images of cocoa leaves and provide accurate, real-time diagnoses, helping farmers take timely and informed actions to protect their crops.

At its core, the system is built using Flask as the backend framework, ensuring a lightweight and scalable web architecture. The disease classification engine is powered by a deep learning model developed with TensorFlow, specifically a DenseNet121 architecture that has been trained on labeled cocoa leaf images. This model was selected for its strong performance and ability to generalize well across different disease patterns, achieving high accuracy in classification tasks.

Users interact with the system through a simple and intuitive interface where they can upload images of affected cocoa leaves. Once an image is submitted, it is processed by the trained model, which extracts relevant features and predicts the most likely disease category. The result is then presented to the user along with basic information about the disease, enabling quicker understanding and response.

To support accessibility and real-world deployment, the application can be exposed to external users using Ngrok, allowing it to run locally while still being reachable via the internet. This is particularly useful during development and testing phases. The model itself is hosted and managed via Kaggle, ensuring easy version control, reproducibility, and integration with training datasets.

Cocoa Care addresses a critical challenge in agriculture—limited access to timely and accurate disease diagnosis, especially in rural farming communities. By digitizing and automating the diagnostic process, the system reduces dependency on manual inspection and agricultural experts, which are often scarce. Ultimately, the project aims to improve crop yield, reduce losses, and promote sustainable cocoa farming practices through the integration of artificial intelligence into agriculture.
